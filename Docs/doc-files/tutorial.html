tutorial.jl

¶

project.csv file can be formatted as

name, sn, tn, L, M, R0, h0, E, Rt

with name::ASCIIString vessel clinical name sn::Int source node ID, tn::Int terminal node ID (both sn and tn refer to the global topological description of the system), L::Float vessel length (m), M::Int number of nodes within the vessel (numerical discretisation), R0::Float unstressed lumen radius (m), h0 vessel wall thickness (m), E vessel wall Youngâ€™s modulus (Pa), Rt outlet reflection coefficient \(R_t \in [-1,1]\).

Outlet can be coupled with a three element windkessel by specifying lumped parameters values

name, sn, tn, L, M, R0, h0, E, R1+R2, Cc

or

name, sn, tn, L, M, R0, h0, E, R1, R2, Cc
